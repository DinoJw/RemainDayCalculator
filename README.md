# DaysUntilTest

올해 내에서 계산 가능합니다.
또한 입력 일정의 오전 9시를 기준으로 계산합니다.

매개변수 :
n = 일정의 이름
m = 일정의 달(month)
d = 일정의 날(day)
e = 이벤트 (혼자 사용, 아래의 링크만 사용 가능합니다)

https://dinojw.github.io/RemainDayCalculator
